Module Error_log.Error

type t

Sometimes you want to be able to catch fatal errors raised by some code, so you may continue to run some logic afterwards. This is what this module is about. The intended usage is to call Error_log.protect and match on the result.

val raise : t -> _
val continue : t -> Base.unit

To be used in place of ignore when pattern matching on Error reported.
